# CoCart Tests

## Description

The purpose of this plugin is to provide developers a helpful tool to test a situation when using CoCart. You can access the API's with or without authentication.

### Tests available

1: User management. Returns true or false statements on the cart session, identify if request is authenticated, the requested cart, if the user is a registered customer or a guest and if the cart can merge item content.
> `GET - /wp-json/cocart/test/user-management`
